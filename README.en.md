## Overview

The second phase of the digital twin pathway: **progressive fidelity growth (1→100)**. This is not about scaling to more twins—it is about making a single twin increasingly similar to its real person over time.

A twin improves not only because the user says more about themselves. It also improves because it **acts, responds, accumulates experience, and is corrected** within the digital world. These behavioral traces become evidence for refinement.

---

## Fidelity Dimensions

| Dimension | Description |
|-----------|-------------|
| Self-report consistency | Stability of self-description across contexts |
| Behavioral consistency | Match between actions and claimed style |
| Stylistic consistency | Vocabulary, tone, syntactic fidelity |
| Relational consistency | Stability of interaction patterns with others |
| Long-term stability | Overall personality stability across time |

**Validation tiers**: Weakly-grounded twins (low-risk assistance) ↔ Highly-calibrated twins (strong social simulation / delegated participation)
